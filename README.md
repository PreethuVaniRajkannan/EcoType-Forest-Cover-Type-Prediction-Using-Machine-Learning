# EcoType-Forest-Cover-Type-Prediction-Using-Machine-Learning
A machine learning classification model was created for predicting the type of forest cover in a given geographical area based on cartographic variables like elevation, soil type, slope, and wilderness area information
The goal of the project is to assist in environmental monitoring, forestry management, and land use planning by providing automated, reliable forest cover type identification.
It is a multi class classification problem with Target Variable: Cover_Type with 7 claseses
Steps followed:
            Data set collection
            Data cleaning - Null values, duplicates, Value counts checked
            Outliers detection - BOX Plot
            Outliers handling - IQR method
            Data Processing - Skew correction
            EDA - univariate and bivariate analysis, correlation analysis
            Feature importances
            Class Imbalance handling - SMOTE method
            Machine learning model building  
                train-test split
                Models checked for accuracy:
                          Logistic Regression
                          Decision Tree classifier
                          Random forest classifier
                          Knn classifier
                          XGboost classifer
                Cross Validation with Best Model: Random forest classifier
                Grid Search CV
                Random Search CV
                Model Fit and Train
                Prediction
                Model saved using Pickling
                Streamlit build for Model prediction visualization with features.
